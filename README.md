# 鲜花花卉商城系统 python364

## 项目概述

鲜花花卉商城系统是基于Python语言和Django框架开发的一套在线购物平台。系统致力于为用户提供便捷的鲜花购买、预订服务，同时为花店提供高效的管理与运营解决方案。

## 技术栈

- **后端**: Python 3.x, Django
- **前端**: HTML, CSS, JavaScript, Bootstrap
- **数据库**: MySQL

## 功能模块

- **用户模块**
  - 用户注册与登录
  - 用户信息管理
  - 购物车管理
  - 订单管理
- **商品模块**
  - 商品展示
  - 商品搜索
  - 商品分类管理
  - 商品评价
- **花店管理模块**
  - 订单处理
  - 商品管理
  - 用户管理
  - 数据统计

## 系统角色

- **普通用户**: 可浏览商品、添加购物车、下单购买。
- **花店管理员**: 管理商品信息、处理订单、监控用户反馈。

## 运行环境

- **操作系统**: Linux或Windows
- **Python版本**: Python 3.x
- **数据库**: MySQL 5.x 或更高版本
- **Web服务器**: 推荐使用 Gunicorn 作为WSGI HTTP服务器

## 部署步骤

1. 安装Python环境
2. 安装依赖包
3. 配置数据库
4. 运行迁移脚本以建立数据库结构
5. 启动Gunicorn服务器
